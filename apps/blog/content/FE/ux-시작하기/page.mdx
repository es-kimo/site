# 시작하며

최근 커리어에 관한 고민이 많았다. `메디컬에이아이`에서 1년간 프론트엔드 개발을 하며, 더 이상 취미가 아닌 일로서 개발을 배우는 값진 시간을 보냈다. 특히 일정 관리와 코드 리뷰 노하우는 얻어가는 것 같다.

우선은 벌려놓은 게 있으니 밀알이라도 회수해보자는 마음으로 1년 더 해보겠다. 돌아오는 해에는 조금 더 명확한 목표를 가져볼 것이다.

- 멋진 애니메이션을 만들 수 있는 개발자
